# Account Sign-In Review Oracle

## Capability

Registered users can sign in to access their dashboard.

## Rules

1. A registered user who submits valid credentials is signed in and can access the dashboard.
2. After three consecutive invalid password attempts, the account is locked.
3. When an account is locked, any further sign-in attempt is rejected with a clear account-locked message.
4. The rules do not define password reset, multifactor authentication, or remember-me behavior.
5. No project tag schema beyond the provided requirement identifier is defined.
